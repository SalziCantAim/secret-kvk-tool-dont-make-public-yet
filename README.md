## Setup

1) Install [taskfile](https://taskfile.dev/installation/)
2) Install [uv](https://docs.astral.sh/uv/getting-started/installation/)

Then run:

    task setup

## How to build

Run the following command:

    task build

## How to run in developers mode (without compiling)

Run the following command:

    task run
